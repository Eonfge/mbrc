MusicBee Remotes requires a plugin to be installed on your computer in order to function. To ensure optimal experience make sure to download the latest version of the plugin.

Note that the version 0.2.5 of MusicBee Remote Plugin is not compatible with the newest versions, so if you are upgrading from version 0.2.3.x of MusicBee Remote make sure to update your plugin to the proper version otherwise the application will not function.

Visit the Download site to get the latest plugin version.
